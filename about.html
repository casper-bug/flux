FLUX

Personal File & Link Synchronization

THE CONCEPT

Flux was built to solve a simple problem: moving a file or a URL from one device to another without using email, Slack, or a privacy-invasive third-party service.

Instead of building another storage silo, Flux turns your existing Google Drive into a personal "drop zone".

HOW IT WORKS

Flux uses your own Google Drive storage. When you upload a file, it's sent directly to a folder called FluxSpace in your Drive. The app uses the drive.file scope, meaning it only has access to the files it creates.

There is no backend server. All logic happens in your browser using the Google Drive API v3.

KEY FEATURES

• Drag & drop files and folders
• Instant link saving and clipboard sync
• PWA support for mobile home screen access
• Offline-first architecture via Service Workers
• Keyed DOM-diff rendering for high performance
